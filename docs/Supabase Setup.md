# Supabase Setup

## What is Supabase?
Supabase provides a managed PostgreSQL database, authentication, and file storage — all from one service. Replaces the custom Python backend + MongoDB + Emergent Object Storage.

## Project
- **Dashboard:** [supabase.com/dashboard](https://supabase.com/dashboard)
- **URL format:** `https://PROJECT_ID.supabase.co`

## Database Tables

### `gallery`
| Column | Type | Default |
|---|---|---|
| id | uuid (PK) | gen_random_uuid() |
| title | text | '' |
| storage_path | text | - |
| content_type | text | 'image/jpeg' |
| size | integer | 0 |
| is_deleted | boolean | false |
| created_at | timestamptz | now() |

### `reviews`
| Column | Type | Default |
|---|---|---|
| id | uuid (PK) | gen_random_uuid() |
| author_name | text | - |
| location | text | null |
| rating | integer | - (check 1-5) |
| text | text | - |
| approved | boolean | false |
| created_at | timestamptz | now() |

### `messages`
| Column | Type | Default |
|---|---|---|
| id | uuid (PK) | gen_random_uuid() |
| name | text | - |
| email | text | - |
| phone | text | - |
| message | text | - |
| read | boolean | false |
| created_at | timestamptz | now() |

## Row Level Security (RLS)

### Public Access (unauthenticated)
- `gallery`: SELECT where `is_deleted = false`
- `reviews`: SELECT where `approved = true`, INSERT (submissions)
- `messages`: INSERT (contact form)

### Admin Access (authenticated)
- Full CRUD on all tables

## Storage
- **Bucket:** `gallery` (public)
- **Upload path:** `gallery/{uuid}.{ext}`
- **Public URL:** auto-generated by Supabase

## Auth
- Uses Supabase Auth (email/password)
- Admin user created in Supabase dashboard (Authentication > Users > Add user)
- Session managed by `onAuthStateChange` listener in `AuthContext.jsx`
