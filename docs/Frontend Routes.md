# Frontend Routes

| Path | Page | Auth Required |
|---|---|---|
| `/` | HomePage | No |
| `/admin/login` | LoginPage | No |
| `/admin` | Overview | Yes |
| `/admin/galerie` | GalleryAdmin | Yes |
| `/admin/recenzii` | ReviewsAdmin | Yes |
| `/admin/mesaje` | MessagesAdmin | Yes |

## Public Sections (HomePage)

The HomePage is composed of these sections:
1. **Header** - Navigation + phone CTA
2. **Hero** - Main banner
3. **Services** - Stamped, helicopter-finished, vertical concrete
4. **About** - Company info
5. **Gallery** - Portfolio images
6. **Reviews** - Client testimonials
7. **Contact** - Contact form + info
8. **Footer** - Links + contact

## Admin Panel

Sidebar layout with sections:
- **Overview** - Dashboard stats
- **Gallery** - Upload/manage images
- **Reviews** - Manage reviews
- **Messages** - View contact form submissions
